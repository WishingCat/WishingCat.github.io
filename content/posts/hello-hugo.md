这是博客的第一篇文章。

我选择 **Hugo + GitHub Pages** 这套方案，主要因为：

1. 生成速度快，内容就是纯文本文件，维护成本低。
2. GitHub Pages 免费，配合 Actions 可以自动部署。
3. 以后想加评论、搜索、自定义域名，也都比较方便。

接下来我会继续补充更多内容，比如：

- Hugo 的写作工作流
- GitHub Pages 自动部署配置
- 我自己的项目记录和学习笔记

如果你正在搭建自己的博客，希望这篇文章能帮你更快开始。
